# Algoritmos Genéticos e suas Implicações Epistemológicas

## Autor
Sávio Arbuês Abrahão Nery  
14-11-2025  

## Curso
Modelagem Matemática Aplicada à Otimização na Análise de Dados  
Disciplina: Otimização Aplicada à Ciência de Dados

## Resumo
Este trabalho analisa criticamente algoritmos genéticos (AG) sob perspectivas epistemológicas, filosóficas e aplicacionais, explorando a tensão entre design inteligente, processos estocásticos, questões teológicas e aplicações em jogos eletrônicos, como Trackmania.
